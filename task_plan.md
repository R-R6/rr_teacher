# 高中化学教学辅助系统 — 开发进度

## Goal

完成高中化学教学辅助系统的 MVP 版本：老师拍照 → 识别出化学题 → 编辑修正 → 一键导出 Word，并搭建微信小程序前端。

**Phase 0 (环境准备):** ✅ 基本完成（SQLite开发环境、Docker配置、微信开发者工具+MCP已装）
