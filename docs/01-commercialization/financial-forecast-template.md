## 七、真实数据获取建议

| 数据 | 获取方式 |
|---|---|
| OCR 单次成本 | 豆包 / pix2text 官方定价 × 实际调用参数 |
| CloudRun 成本 | 腾讯云账单 × 预估日活 / 1000 |
| COS 成本 | 图片数量 × 平均大小 × 存储单价 + 下载流量单价 |
| 老师数量 / 付费意愿 | 教师社群问卷、小红书/公众号后台、种子用户访谈 |
| 获客成本 | 小规模投放测试：花费 / 注册数 / 付费数 |

---

## 八、使用说明

1. 先填写“核心假设”表格，所有下游表格尽量由假设推导而来。
2. 第 1 年建议按月度拆分，第 2–3 年可按季度或年度汇总。
3. 单位经济模型中的 CAC 和 LTV 是投资人最关心的指标，建议单独做敏感性分析。
4. 所有占位符替换后，可导出到 Excel 或 Google Sheets 生成图表。
